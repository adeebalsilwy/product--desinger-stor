# دليل الاستخدام الكامل - صفحة المصمم

## 🎨 مقدمة

تم تحديث صفحة المصمم (Designer) لتطابق تصميم صفحة PHP مع إضافة ميزات احترافية شاملة.

---

## ✨ الميزات الجديدة

### 1. **أزرار الإغلاق للنماذج**
كل نموذج (Palette) الآن يحتوي على زر إغلاق (X) في الزاوية العلوية اليمنى.

**النماذج المتاحة:**
- 🎨 لوحة الألوان
- 📏 لوحة مقاس الأداة  
- 👗 لوحة مقاسات الفستان
- 🌈 لوحة ألوان الفستان
- 📐 لوحة القوالب
- 📝 لوحة النص
- 🖼️ لوحة الصورة
- ⚙️ لوحة الخصائص

### 2. **نظام الطبقات (Layers System)**

```
┌─────────────────────────────────┐
│    Elements Layer (z-index: 3)   │ ← النصوص، الصور، الأشكال
├─────────────────────────────────┤
│    Drawing Layer (z-index: 2)    │ ← الرسم الحر
├─────────────────────────────────┤
│  Template Layer (z-index: 1)     │ ← خلفية الفستان
└─────────────────────────────────┘
```

**كيفية الترتيب:**
1. حدد العنصر
2. استخدم "للأمام" أو "للخلف" من لوحة الخصائص
3. أو عدل `zIndex` مباشرة

### 3. **أدوات التعديل الشاملة**

#### 📐 تغيير الحجم (Resize)
- مقبض في الزاوية اليمنى السفلى
- اسحب المقبض لتغيير الحجم
- يظهر البعد الحالي (عرض × ارتفاع) أعلى العنصر

#### 🔄 الدوران (Rotate)
- مقبض في أعلى العنصر
- دوران حر 360 درجة
- مؤشر بصري للاتجاه

#### 📏 مؤشر المقياس
- يعرض الأبعاد بالبكسل
- يتحدث فورياً أثناء التغيير
- مثال: `250 × 180`

### 4. **المجسم الأنثوي المحسن**

```
نسب الجسم:
├── الرأس: 75px × 92px
├── الرقبة: 26px × 34px  
├── الجذع: 170px × 290px
├── الأرداف: 185px
└── الأرجل: 50px × 175px
```

**المميزات:**
- ✅ جسم أنثوي متناسق
- ✅ خصر نحيف وأرداف عريضة
- ✅ أكتاف وأذرع طبيعية
- ✅ دوران 360 درجة سلس
- ✅ ظل واقعي على الأرضية

## 🛠️ الأدوات وطريقة الاستخدام

### 1. **الفرشاة (Brush)**
```javascript
// خصائص الفرشاة:
- الأحجام: 1px إلى 50px
- الألوان: 15 لون جاهز
- الرسم: حر وناعم
```

**الاستخدام:**
1. اضغط على أيقونة الفرشاة
2. اختر اللون من لوحة الألوان
3. اختر حجم الفرشاة
4. ابدأ الرسم على canvas

### 2. **القلم (Pen)**
```javascript
// خصائص القلم:
- أدق من الفرشاة
- مثالي للتفاصيل الدقيقة
- سمك افتراضي أقل
```

### 3. **الممحاة (Eraser)**
```javascript
// خصائص الممحاة:
- مسح شفاف (destination-out)
- نفس أحجام الفرشاة
- تعيد الخلفية البيضاء
```

### 4. **الألوان (Colors)**
```
لوحة الألوان:
├── أسود، أحمر، أخضر، أزرق
├── أصفر، وردي، سماوي، برتقالي
├── بنفسجي، فوشيا، أزرق فاتح
└── أبيض، رمادي فاتح، رمادي غامق
```

### 5. **مقاس الأداة (Tool Size)**
```
الأحجام المتاحة:
1px, 3px, 5px, 8px, 10px, 15px, 20px, 30px, 50px
```

### 6. **المقاسات (Sizes)**
```
مقاسات الملابس:
XS, S, M, L, XL, XXL
```

### 7. **لون الفستان (Dress Color)**
```
15 لون باستيل جاهز:
- أبيض، وردي فاتح، بنفسجي فاتح
- أزرق فاتح، أخضر فاتح، بيج
- وغيرها من الألوان الهادئة
```

### 8. **القوالب (Templates)**
```
القوالب المتاحة:
├── هادئ (خلفية بيضاء ناعمة)
├── نيون موف (تدرج بنفسجي)
├── شبكي (نمط هندسي)
└── فاخر (تدرج دائري)
```

### 9. **النص (Text)**
```
خصائص النص:
├── المحتوى: نص مخصص
├── حجم الخط: 12px - 120px
├── وزن الخط: عادي، متوسط، عريض، ثقيل
└── اللون: أي لون
```

**إضافة نص:**
1. اضغط على أداة النص
2. اكتب النص في المستطيل
3. اضغط "إضافة"
4. عدل الخصائص من Properties Palette

### 10. **الأشكال (Shapes)**
```
الأشكال المتاحة:
├── مستطيل (Rect)
├── دائرة (Circle)
└── مثلث (Triangle)
```

**إضافة شكل:**
1. اضغط على أداة الأشكال
2. سيتم إضافة شكل تلقائياً
3. اسحب لتغيير الموقع والحجم
4. عدل الخصائص من اللوحة

### 11. **الصورة (Image)**
```
支持的图片格式:
- JPG/JPEG
- PNG
- GIF
- WebP
```

**رفع صورة:**
1. اضغط على أداة الصورة
2. اضغط "تصفح"
3. اختر صورة من جهازك
4. ستظهر على canvas

### 12. **التحديد (Select)**
```
وظائف التحديد:
├── تحديد العناصر
├── تحريك العناصر
├── تغيير الحجم
└── تعديل الخصائص
```

## 🎯 خطوات العمل الكاملة

### مثال: تصميم فستان كامل

#### الخطوة 1: إعداد الخلفية
```
1. اختر "لون الفستان" → لون وردي (#f8bbd0)
2. اختر "القوالب" → قالب "هادئ"
3. اختر "المقاسات" → M
```

#### الخطوة 2: رسم زخرفة
```
1. اختر "فرشاة" → حجم 5px
2. اختر "ألوان" → ذهبي (#ffeb3b)
3. ارسم زخرفة على الفستان
```

#### الخطوة 3: إضافة نص
```
1. اختر "نص" → اكتب "أحلام غيرلز"
2. اضغط "إضافة"
3. اسحب النص لمكان مناسب
4. عدل حجم الخط إلى 60px
5. غير الوزن إلى "عريض"
```

#### الخطوة 4: إضافة شعار
```
1. اختر "صورة" → "تصفح"
2. اختر ملف الشعار (logo.png)
3. عدل الحجم إلى 150×150
4. ضع الشعار في الزاوية
```

#### الخطوة 5: ترتيب الطبقات
```
1. حدد النص
2. اضغط "للأمام" مرتين
3. حدد الشعار
4. اضغط "للأمام" مرة
```

#### الخطوة 6: المعاينة والحفظ
```
1. اضغط "عرض التصميم على المجسم"
2. شاهد التصميم على المجسم الأنثوي
3. اضغط "تدوير المجسم" للدوران
4. اضغط "تنزيل" لحفظ التصميم
```

## 🎨 نصائح تصميم احترافية

### 1. **اختيار الألوان**
- استخدم ألوان متناسقة (Color Harmony)
- اختر ألوان باستيل للخلفيات
- استخدم ألوان زاهية للتفاصيل

### 2. **الطبقات**
- رتب العناصر منطقياً
- الخلفية أولاً ثم التفاصيل
- اجعل النصوص في الأعلى

### 3. **الحجم والموقع**
- استخدم قاعدة الثلث في التوزيع
- لا تزدحم العناصر
- اترك مساحات بيضاء

### 4. **النصوص**
- اختر خطوط واضحة
- حجم مناسب للقراءة
- تباين جيد مع الخلفية

## 🔧 حل المشاكل

### المشكلة: الرسم لا يظهر
**الحل:**
1. تأكد من اختيار الأداة الصحيحة
2. تحقق من أن اللون ليس شفاف
3. جرب تغيير حجم الفرشاة

### المشكلة: العناصر لا تتحرك
**الحل:**
1. حدد الأداة "تحديد"
2. اضغط على العنصر لتحديده
3. اسحب بحركة سلسة

### المشكلة: الحفظ لا يعمل
**الحل:**
1. تأكد من اكتمال التصميم
2. انتظر ثواني بعد آخر تعديل
3. جرب تحديث الصفحة
