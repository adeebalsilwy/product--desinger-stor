# محرر الصور الاحترافي للمجسمات - دليل الاستخدام الكامل

## 🎯 نظرة عامة
تم تطبيق نظام احترافي كامل لتعديل الصور والتصاميم على المجسمات ثلاثية الأبعاد مع واجهة نظيفة ومنظمة ودعم كامل للغتين العربية والإنجليزية.

---

## ✨ المميزات الرئيسية

### 1️⃣ **نافذة منبثقة احترافية (Modal)**
نافذة كاملة تظهر عند الضغط على زر الأدوات البنفسجي، تحتوي على:

#### **أقسام النافذة:**

**📌 قسم اختيار الصور (أزرق)**
- شبكة عرض للصور المتاحة
- تحديد الصورة الحالية بإطار ذهبي
- 4 أعمدة للجوال، 6 أعمدة للكمبيوتر

**⚡ الإجراءات السريعة (أخضر)**
أربعة أزرار أساسية:
1. **إزالة الخلفية** - تشغيل/إيقاف مع أيقونة الممحاة
2. **التدوير التلقائي** - بدء/إيقاف دوران المجسم
3. **إعادة ضبط التصميم** - حذف جميع التعديلات
4. **إعادة الكاميرا** - العودة للوضع الافتراضي

**🎯 التحكم بالموضع (بنفسجي)**
أزرار الاتجاهات:
- ↑ للأعلى
- ↓ للأسفل
- ← لليسار
- → لليمين

**📏 الحجم والدوران (برتقالي)**
أزرار التحويل:
- ➕ تكبير
- ➖ تصغير
- ↺ تدوير لليسار
- ↻ تدوير لليمين

**🎚️ الضبط الدقيق (نيلي)**
شرائح تمرير متقدمة مع عرض القيم:
- **دوران Z**: -180° إلى 180°
- **دوران X**: -45° إلى 45°
- **الانحناء**: 0% إلى 25%
- **الشفافية**: 20% إلى 100%
- **مقياس X**: 0.2 إلى 2.0
- **مقياس Y**: 0.2 إلى 2.0
- **محور X**: -1.0 إلى 1.0
- **محور Y**: -0.5 إلى 1.5
- **محور Z**: -0.2 إلى 1.0

---

### 2️⃣ **واجهة العرض المبسطة**

#### **قبل:**
- أزرار كثيرة في الشريط العلوي
- واجهة معقدة

#### **بعد:**
- **واجهة نظيفة وبسيطة**
- الشريط العلوي يحتوي فقط على:
  - زر المجسم السابق/التالي
  - زر إظهار/إخفاء التصميم
- أزرار الوصول السريع على اليمين:
  - أيقونة الخصائص (ترس)
  - زر أدوات الصورة (بنفسجي)

---

### 3️⃣ **لوحة خصائص العرض**
عند تفعيلها، تظهر لوحة سفلية بالتفاصيل:

كل مقياس بلون مختلف:
- 🔵 أزرق - الموضع (X, Y, Z)
- 🟢 أخضر - الحجم (X, Y)
- 🟣 بنفسجي - الدوران (X, Y, Z درجات)
- 🟠 برتقالي - الانحناء نسبة
- 🩷 وردي - الشفافية نسبة
- 🔷 نيلي - الجودة (عالية/قياسية)

## 🛠️ كيفية الاستخدام

### **للمستخدمين:**

#### **الطريقة 1: الإعداد السريع**
1. افتح نافذة المحرر بالضغط على الزر البنفسجي
2. اختر صورة من الشبكة
3. اضغط "إزالة الخلفية" إذا لزم
4. استخدم أزرار الأسهم لضبط الموضع
5. اضغط "تم" عند الانتهاء

#### **الطريقة 2: الضبط الدقيق**
1. افتح لوحة الخصائص
2. استخدم شرائح التمرير للضبط الدقيق
3. راقب القيم الدقيقة في اللوحة
4. اختبر على مجسمات مختلفة
5. احفظ عند الرضا

#### **الطريقة 3: إعادة البدء**
1. اضغط "إعادة ضبط التصميم"
2. اختر صورة جديدة
3. يتم الضبط تلقائياً للوضع الافتراضي
4. قم بتعديلات بسيطة
5. انتهى

## 💡 نصائح احترافية

1. **ابدأ بالإجراءات السريعة**: استخدم الأزرار المعدة مسبقاً
2. **استخدم الضبط الدقيق**: للحصول على دقة البكسل
3. **تحقق من لوحة الخصائص**: راقب القيم الدقيقة
4. **اختبر على مجسمات مختلفة**: التصاميم تبدو مختلفة على الأجسام المختلفة
5. **أعد الضبط غالباً**: إذا ضعت، أعد الضبط وابدأ من جديد
6. **اجمع الحركات**: استخدم تعديلات متعددة معاً
7. **استخدم إزالة الخلفية**: للتصاميم الشفافة

## 📱 الاستجابة للشاشات

### **كمبيوتر (>1024px):**
- نافذة كاملة بجميع الأقسام
- شبكة صور 6 أعمدة
- أزرار الإجراءات 4 أعمدة
- شرائح جانباً جنب

### **تابلت (768px - 1024px):**
- نافذة قابلة للتمرير
- شبكة صور 4 أعمدة
- أزرار الإجراءات عمودين
- شرائح متراكبة

### **جوال (<768px):**
- نافذة مضغوطة
- شبكة صور 4 أعمدة
- أزرار الإجراءات عمودين
- عمود واحد للشرائح
- إخفاء العناوين الصغيرة

## 🔧 الوظائف البرمجية

### **الدوال المتاحة:**

```javascript
// فتح/غلق النافذة
openImageEditorModal()
closeImageEditorModal()

// تبديل لوحة الخصائص
togglePropertyOverlays()

// التحكم بالدوران
setAutoRotate(enabled)
toggleAutoRotate()

// تحميل صورة مع إزالة الخلفية
loadProductTextureWithBackgroundRemoval(url, reset)

// تحويل التصميم
rotateDesignLeft()
rotateDesignRight()
scaleDesignUp()
scaleDesignDown()
moveDesignUp()
moveDesignDown()
moveDesignLeft()
moveDesignRight()
resetDesignTransform()
resetView()
```

### **الخصائص:**

```javascript
showImageEditorModal // رؤية النافذة
showPropertyOverlays // رؤية لوحة الخصائص
isBackgroundRemovalActive // حالة إزالة الخلفية
projection // { x, y, z, scaleX, scaleY, rotateX, rotateY, rotateZ, curve, opacity }
```
