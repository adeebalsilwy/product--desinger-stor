   - التوثيق الآن في `docs/`
   - النصوص الآن في `scripts/`

2. **تحديث مراجع النصوص**
   ```bash

   php check-settings.php
   
   # جديد
   php scripts/check-settings.php
   ```

3. **تحديث روابط التوثيق**
   ```markdown
   # قديم
   انظر IMPLEMENTATION_GUIDE.md
   
   # جديد
   انظر docs/IMPLEMENTATION_GUIDE.md
   ```

### لأعضاء الفريق الجدد

1. ابدأ بـ `README.md` في الجذر

3. ابحث عن الأدلة في `docs/`
4. استخدم النصوص من `scripts/`
