# 🎨 دليل الاستخدام السريع - أدوات الفرشاة والألوان

## 📋 الوصول السريع

### فتح استوديو الفرشاة
1. انقر على أداة **✏️ Brush** في شريط الأدوات
2. ستظهر لوحة الفرشاة على اليمين تلقائياً

### إغلاق اللوحة
- انقر على زر **[✕]** في أعلى اللوحة
- أو اختر أداة أخرى من شريط الأدوات

---

## 🖌️ استخدام الفرشاة

### الخطوات الأساسية
```
1. اختر نوع الفرشاة ✏️💨🖊️🖌️🧽
2. حدد اللون من palette 🎨
3. اضبط الحجم 1-100px
4. اضبط الشفافية 0-100%
5. ابدأ الرسم! 🎨
```

### أنواع الفراشي وسرعتها

| النوع | السرعة | الاستخدام الأمثل |
|-------|--------|------------------|
| ✏️ Pencil | سريعة | الخطوط الدقيقة، التفاصيل |
| 💨 Spray | متوسطة | التظليل، التدرجات |
| 🖊️ Marker | سريعة | التحديد، الخطوط العريضة |
| 🖌️ Soft | بطيئة | الرسم الحر، المزج |
| 🧽 Eraser | سريعة | التصحيح، الحذف |

## 🎨 اختيار الألوان

### التبويبات الأربعة

#### 1️⃣ Basic (الأساسية)
- **19 لون أساسي**
- ألوان ابتدائية وثانوية
- مثالية للرسم السريع

**الألوان المتاحة:**
```
⚫ أسود     ⚪ أبيض      🔴 أحمر
🟢 أخضر    🔵 أزرق      🟡 أصفر
🟣 بنفسجي  🩷 وردي      🤎 بني
```

#### 2️⃣ Advanced (متقدمة)
- **30 لون متقدم**
- درجات متنوعة
- للرسومات التفصيلية

**أمثلة:**
```
🩷 وردي فاتح   🥇 ذهبي       🍅 طماطم
🌊 فيروزي      🌸 خزامي      🪨 صخري
```

#### 3️⃣ Gradients (تدرجات)
- **10 تدرجات لونية**
- انتقالات جميلة
- للخلفيات والتأثيرات

**التدرجات:**
```
أحمر ← أصفر   (غروب)
وردي ← سماوي  (حلم)
أخضر ← أزرق   (طبيعة)
```

#### 4️⃣ Metallic (معدنية)
- **8 ألوان معدنية**
- تأثيرات لامعة
- للفخامة والزينة

**المعادن:**
```
🥇 ذهبي      🥈 فضي       🥉 برونزي
✨ بلاتين    🏆 ذهب قديم  💿 فضي فاتح
```

## ⚙️ الضوابط والإعدادات

### حجم الفرشاة
```
الشريط: ═══●═══════
القيمة: 1px ── 100px
النصيحة: احفظ الماوس واسحب للتغيير السريع
```

### الشفافية
```
الشريط: ════●══════
القيمة: 0% ── 100%
مثالي: 50-80% للطبقات، 100% للألوان الصلبة
```

### معاينة الحجم
- انظر إلى الدائرة أسفل شريط الحجم
- تمثل الحجم الفعلي قبل الرسم
- تتغير باللون المحدد

## 🎯 نصائح احترافية

### للرسم الدقيق
✅ استخدم Pencil (✏️)
✅ حجم 1-5px
✅ شفافية 100%
✅ ألوان Basic

### للتظليل
✅ استخدم Spray (💨) أو Soft (🖌️)
✅ حجم 20-50px
✅ شفافية 30-50%
✅ تدرجات لونية

### للمسح
✅ استخدم Eraser (🧽)
✅ حجم متوسط 15-30px
✅ اضغط بقوة للمسح الكامل

### للطبقات
✅ ابدأ بشفافية منخفضة
✅ زد الشفافية تدريجياً
✅ استخدم ألوان متوافقة

## ⌨️ الاختصارات

### أثناء الرسم
```
Ctrl + Z  = تراجع (Undo)
Ctrl + Y  = إعادة (Redo)
Esc       = إلغاء التحديد
Delete    = حذف العنصر المحدد
```

### التنقل
```
الماوس + سحب = تحريك اللوحة
Scroll       = تكبير/تصغير الحجم
Shift + Scroll = تحريك أفقي
```

## 🎨 خلط الألوان

### تقنيات متقدمة

#### 1. Layering (الطبقات)
```
الخطوة 1: لون أساسي (100%)
الخطوة 2: لون ثانوي (50%)
الخطوة 3: مزج برفق
```

#### 2. Gradient Blend (مزج متدرج)
```
اختر لونين متقاربين
ابدأ بالأغمق
انتقل للأفتح تدريجياً
استخدم Spray للمزج
```

#### 3. Metallic Accent (إضافة معدني)
```
ارسم باللون العادي
أضف لمسات معدنية
ركز على النقاط البارزة
اجعلها خفيفة
```

## ❓ حل المشاكل

### المشكلة: الفرشاة لا ترسم
```
الحل:
1. تأكد من تحديد أداة الفرشاة
2. تحقق من الشفافية (ليست 0%)
3. جرب تغيير اللون
4. أعد تحميل الصفحة
```

### المشكلة: اللون غير صحيح
```
الحل:
1. تحقق من التبويب النشط
2. أعد اختيار اللون
3. امسح الكاش
4. جرب متصفح آخر
```

### المشكلة: الحجم لا يتغير
```
الحل:
1. حرك الشريط ببطء
2. تحقق من القيمة المعروضة
3. جرب الكتابة مباشرة
4. أعد تحميل الصفحة
```
